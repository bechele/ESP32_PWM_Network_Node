## This firmware is intended for the ESP32_PWM_NODE_PCB.
It manages to receive broadcast data packets from the bechele software https://github.com/bechele/bechele and controls servos and relais connected to the ESP32_PWM_NODE_PCB.
The firmware needs the enclosed Library that derives from:
https://github.com/ramadhanafif/Ethernet2-W5500-ESP32.
For details read the documentation in folder Documentation and/or visit my web site: http://bechele.de
